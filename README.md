## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/en/)
- [git](https://git-scm.com/)

### Installation

1. **Clone the repo**

    ```sh
    git clone
    ```

2. **Install NPM packages**

    ```sh
    npm i
    ```

    or

    ```sh
    yarn install
    ```

    or

    ```sh
    pnpm i
    ```

3. **Start the development server**

    ```sh
    npm run dev
    ```

    or

    ```sh
    yarn dev
    ```

    or

    ```sh
    pnpm dev
    ```

4. **Open [http://localhost:5173](http://localhost:5173) with your browser to see the result.**
